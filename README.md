# pear2pear (i like pears)

A simple peer-to-peer transfer application written in Java

**Installation:**

```bash
git clone https://github.com/Tjabooo/pear2pear.git
cd pear2pear/src
```

**Usage:**

On **Linux/MacOS:**

```bash
java pear2pear/Pear.java [server|client <host>]
```

On **Windows:**

no idea

**Example usage:**

Server-side
```bash
java pear2pear/Pear.java server
...follow instructions
```

Client-side
```bash
java pear2pear/Pear.java client <host>
...follow instructions
```
host is the IP shown when running the server. very safe with many protection layers yes yes.
